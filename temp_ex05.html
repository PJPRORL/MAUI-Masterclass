                          # Oefening Games

                          Maak een model Game met volgende properties:

                          • int Id
                          • string Naam
                          • int moeilijkheidsgraad (1 tot 5)
                          • string uitgever. Maak een map Data met daarin de mappen IRepositories en Repositories. Maak een Interface en Repository voor het model Game. Registreer de Interface in MauiProgram.cs. Neem deze gegevens over (Copy â€“ Paste):
                          public class GameRepository
 {
     private List<Game> Games = new()
 {
     new Game() { Id = 1, Naam = "Cluedo", Moeilijkheidsgraad = 2, Uitgever = "Hasbro Gaming" },
     new Game() { Id = 2, Naam = "Monopoly", Moeilijkheidsgraad = 3, Uitgever = "Parker" },
     new Game() { Id = 3, Naam = "Machiavelli", Moeilijkheidsgraad = 3, Uitgever = "999 Games" },
     new Game() { Id = 4, Naam = "Catan", Moeilijkheidsgraad = 4, Uitgever = "999 Games" },
     new Game() { Id = 5, Naam = "Carcassonne", Moeilijkheidsgraad = 4, Uitgever = "999 Games" },
     new Game() { Id = 6, Naam = "SpeakOut", Moeilijkheidsgraad = 5, Uitgever = "Hasbro Gaming" },
     new Game() { Id = 7, Naam = "Skyjo", Moeilijkheidsgraad = 1, Uitgever = "Magilano" },
  };

     // Methode om de werknemers op te vragen
     public List<Game> GetGames()
     {
         return Games;
     }
 }

                          Maak volgende View met bijhorend ViewModel (Onvolledige code voor GamePage vind je onderaan deze opdracht):

                          Gebruik een grid met 2 kolommen en 2 rijen. Haal de lijst van Games op uit de GameRepository en toon deze in de linkse kolom. Maak een Stacklayout in de rechtse kolom met alle properties van het model Game. Gebruik een slider van 1-5 om de moeilijkheidsgraad in te geven.

                          Als je een game ingeeft, wordt het nieuwe game toegevoegd aan de lijst met games. Als je een item selecteert links, worden de gegevens rechts getoond. Als je op Nieuw Game klikt, worden de velden terug leeggemaakt.

                          Onvolledige code voor de View GamePage:
